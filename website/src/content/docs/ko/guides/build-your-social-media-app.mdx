이 가이드에서는 Actionbase를 사용하여 간단한 소셜 미디어 애플리케이션을 구축하는 과정을 안내합니다. **팔로우**, **좋아요**, **피드**와 같은 핵심 기능을 위한 활동 데이터를 모델링하고 제공하는 방법을 배우게 됩니다.

## 구축할 내용 {#what-you-will-build}

이 가이드를 완료하면 다음과 같은 기능을 구현하게 됩니다:

- **팔로우** - 사용자가 다른 사용자를 팔로우할 수 있습니다
- **좋아요** - 사용자가 게시물을 좋아요할 수 있습니다
- **피드** - 팔로우한 사용자의 게시물을 실시간 좋아요 수와 함께 표시합니다

이러한 기능은 대부분의 소셜 애플리케이션에서 사용되는 핵심 패턴을 보여줍니다.

## 사전 요구사항 {#prerequisites}

- Docker
- 웹 브라우저

## 인터랙티브 가이드 시작하기 {#start-the-interactive-guide}

이 가이드에는 로컬에서 실행되는 인터랙티브 구성 요소가 있습니다.

<Steps>

1. **Docker로 Actionbase 시작하기**

   ```bash
   docker run -it -p 9300:9300 ghcr.io/kakao/actionbase:standalone
   ```

2. **인터랙티브 가이드 시작하기**

   CLI 프롬프트(`actionbase>`)가 나타나면:

   ```
   guide start hands-on-social
   ```

3. **화면 레이아웃**

   세 개의 패널이 있는 화면을 볼 수 있습니다:
   - **왼쪽**: 가이드 단계를 보여주는 진행 상황 사이드바
   - **중앙**: 소셜 앱 UI
   - **오른쪽**: CLI 터미널 및 API 로그

   팝업 창이 각 단계를 안내해 드립니다. 문제가 발생하면 왼쪽 상단 모서리에 있는 <span class="icon-row"><RefreshCcw size={14} /> **다시 시작**</span>을 클릭하세요.

   ![화면 레이아웃](/ko/images/guides/social-media/00-screen-layout.png)

4. **브라우저에서 열기**

   ```
   http://localhost:9300
   ```

   아래 섹션에서는 보게 될 내용을 요약합니다. 가이드를 실행하지 않고도 읽어볼 수 있으며, 따라 진행하는 동안 참고 자료로 사용할 수 있습니다.

</Steps>

여기서부터는 **웹 브라우저**에서 계속 진행하세요. 인터랙티브 가이드가 각 단계를 안내해 드립니다.

<Steps>

1.  **환영**

    Actionbase 실습 가이드에 오신 것을 환영합니다! 이 튜토리얼에서는 다음을 배우게 됩니다:

    **Tip:** Press **Enter** to proceed.

    ![Zipdoki 소개](/ko/images/guides/social-media/02-zipdoki-intro.png)

3.  **설정**

    먼저, 샘플 데이터를 로드하여 구축에 집중할 수 있도록 하겠습니다.

        **@zipdoki**가 **@j4rami**를 팔로우하도록 합니다. 이는 두 사용자 간의 연결을 생성합니다.

        <ul class="icon-rows">

            <Zap size={18} /> Precomputing count & index

        ```bash
        mutate user_follows --type INSERT --source zipdoki --target j4rami ...
        ```

        <details>
        <summary>curl</summary>

        ```bash

            -H "Content-Type: application/json" \
            -d '{
              "mutations": [{
                "type": "INSERT",
                "edge": {
                  "version": 1737849600000,
                  "source": "zipdoki",
                  "target": "j4rami",
                  "properties": { "createdAt": 1737849600000 }

            }'
        ```

## 요약 {#summary}

방금 팔로우와 좋아요로 피드를 구축했습니다. 이 모든 것이 Actionbase로 구동됩니다.

### 학습 내용 {#what-you-learned}

| Concept            | Description                                                    |
| ------------------ | -------------------------------------------------------------- |
| **Tables**         | Create tables with schemas, indexes, and bidirectional queries |
| **Mutations**      | Insert edges representing user interactions                    |
| **Queries**        | Get individual edges, count totals, and scan with indexes      |
| **Precomputation** | Counts and indexes are computed at write time for fast reads   |
