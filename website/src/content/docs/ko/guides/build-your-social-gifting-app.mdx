이 가이드에서는 Actionbase의 **멀티 엣지** 기능을 활용하여 소셜 커머스 선물 기능을 구축하는 방법을 설명합니다.

## 구현할 기능 {#features-to-implement}

- 선물 발송 및 수신 내역
- 동일한 사용자 간 여러 선물 전송
- 선물 통계 및 추천

## Actionbase 기능: 멀티 엣지 {#actionbase-capability-multi-edge}

기본적으로 Actionbase의 엣지는 **유니크 엣지**입니다. 즉, (source, target) 쌍마다 하나의 엣지로 식별됩니다. 멀티 엣지는 이 모델을 확장한 것입니다:

- **유니크 엣지**: (source, target) 쌍마다 하나의 엣지
- **멀티 엣지**: (source, target) 쌍마다 여러 엣지를 허용하며, 각 엣지는 고유한 `id`로 식별됩니다.

멀티 엣지는 동일한 엔티티 간에 여러 상호작용을 추적해야 할 때 유용합니다. 예를 들어, 동일한 사용자가 동일한 수신자에게 여러 번 선물을 보낼 수 있는 선물 기록과 같은 경우에 사용됩니다.
